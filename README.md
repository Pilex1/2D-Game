# 2D-Game
A 2D side scrolling game created in C# with OpenGL.

Features:
- procedural terrain generation with varying biomes
- dynamic terrain interaction
- logic system
- entities
- realtime, realistic, coloured lighting
- fluid system
- saving and loading worlds
